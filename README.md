# Cloudli

<img src="cloudli.svg" style="zoom: 25%;" />

## Features

- Raspberry pi 5
- Connects via USB-C as Thumbdrive to any device
- [Cooled Case](https://www.galaxus.ch/de/s1/product/elecrow-mini-pc-case-mit-oled-display-fuer-raspberry-pi-5-entwicklungsboard-kit-64719636)
- syncs with nextcloud via WIFI using a dedicated User
- hardware toggle between sync-mode and thumbdrive mode
- [1TB M.2 2280 SSD](https://www.galaxus.ch/de/s1/product/coreparts-m2-sata-iii-2280-1000-gb-m2-2280-ssd-18170636) encrypted with LUKS and unlocked with Yubikey
- shuts down when nextcloud is 100% synced and device has no activity for 30 mins
- Status-OLED: (1)WiFi Connected, (2) 100% Synced, (3) Decrypted, (4) Free Space
- config backend to select folders and timeout
